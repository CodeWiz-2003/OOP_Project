\doxysubsection{Detailed Description}
\begin{DoxyVerb}The game_over_scene class represents the game over scene. It displays the game over message and the final score.

Attributes:
-----------
screen : pygame.Surface
    The main screen surface where the game over scene is displayed.
score : int
    The final score of the game.
font : pygame.font.Font
    The font used to render the text.
running : bool
    A flag indicating whether the game is running.
game_over : bool
    A flag indicating whether the game is over.
\end{DoxyVerb}
